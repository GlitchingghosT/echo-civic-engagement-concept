# Echo — Civic Engagement Concept

A responsive civic-engagement landing-page concept for Ekiti, presenting citizen feedback channels, community initiatives, government updates, and partner content.

- **Live demo:** https://echo-bishop1.vercel.app
- **Source:** https://github.com/GlitchingghosT/echo-civic-engagement-concept

## Project scope

- Responsive desktop and mobile presentation
- Civic information and local-initiative sections
- Static partner and social-media content
- Tailwind-based layout and styling

## Built with

HTML and Tailwind CSS.

## Run locally

No build step is required. Clone the repository and open `index.html` in a browser. Tailwind is loaded through the CDN.
